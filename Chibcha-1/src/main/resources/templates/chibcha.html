¿Qué es un Hosting?

Todos los sitios web constan de un nombre de dominio (www.chibchaweb.com) y un proveedor de alojamiento web (ChibchaWeb). El hosting es el proceso de usar un servidor par guardar un sitio web. En ChibchaWeb puedes tanto obtener los dominios como guardar tu sitio web.

¿Como recupero mi contraseña?

Para recuperar la contraseña debes ir a la página de logueo y oprimir se me olvidó la contraseña. Sigue las instrucciones y luego revisa tu correo para recuperarla.

¿Cuánto tiempo se demora el proceso?

La solicitud de dominio se demora máximo 48 horas. Una vez tengas el dominio y estés dentro de un plan puedes empezar a subir tu sitio web.
